6.1 Coin Katas

6.2 Coin Kata (difficulty = 1)

read
Make a survival game with an avatar and a basic collectable coin.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:coin-list (list (custom-coin)))

6.3 Coin Kata (difficulty = 2)

Tip: The default value of custom-coin is 10.
read
Make a survival game with an avatar and a collectable coin worth 500.
code
#lang survival
 
(define (my-coin) (custom-coin #:value 500))
 
(survival-game
  #:avatar (custom-avatar)
  #:coin-list (list (my-coin)))

6.4 Coin Kata (difficulty = 2)

read
Make a survival game with an avatar and 20 silver coins worth 500.
code
#lang survival
 
(define (my-coin)
   (custom-coin
    #:sprite silver-coin-sprite
    #:name "Silver Coin"
    #:value 500
    #:amount-in-world 20))
 
(survival-game
  #:avatar (custom-avatar)
  #:coin-list (list (my-coin)))

6.5 Coin Kata (difficulty = 3)

read
Make a survival game with an avatar, a silver coin, and a single epic gold coin that never respawns and is worth 1000.
code
#lang survival
 
(define (my-coin)
   (custom-coin
    #:sprite silver-coin-sprite
    #:name "Silver Coin"))
 
(define (my-special-coin)
   (custom-coin
    #:sprite gold-coin-sprite
    #:name "Gold Coin"
    #:value 1000
    #:amount-in-world 1
    #:respawn? #f))
 
(survival-game
  #:avatar (custom-avatar)
  #:coin-list (list (my-coin) (my-special-coin)))

6.6 Sky Katas

6.7 Sky Kata (difficulty = 1)

Tip: By default, the length-of-day will be 2400.
read
Make a survival game with an avatar and a slow changing sky.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:sky (custom-sky #:length-of-day 5000))

6.8 Sky Kata (difficulty = 1)

Tip: Max darkness takes a value between 0 (no darkness) and 255 (total blackout).
read
Make a survival game with an avatar and a fast changing sky with max-darkness.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:sky (custom-sky
   #:length-of-day 500
   #:max-darkness 255))

6.9 Sky Kata (difficulty = 1)

read
Make a survival game with an avatar and a darkmagenta night-sky with 150 darkness.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:sky (custom-sky
   #:night-sky-color 'darkmagenta
   #:max-darkness 150))

6.10 Sky Kata (difficulty = 1)

read
Make a survival game with an avatar, a very short night-time, and an enemy that only comes out at night.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:enemy-list (list
   (custom-enemy
    #:amount-in-world 20
    #:night-only? #t))
  #:sky (custom-sky
   #:length-of-day 2400
   #:start-of-daytime 200
   #:end-of-daytime 2200))

6.16 Bg Kata (difficulty = 2)

Tip: start-tile can take a number between 0 (the default) and 9.
read
Make a survival game with an avatar and a lava custom-bg where the avatar starts on tile 4.
code
#lang survival
 
(define (my-bg)
   (custom-bg #:image LAVA-BG #:start-tile 4))
 
(survival-game
  #:avatar (custom-avatar)
  #:bg (my-bg))

code

6.20 Npc Kata (difficulty = 2)

Tip: All options for #:mode are: ’follow, ’wander, ’pace and ’still.
read
Make a survival game with an avatar and a witch npc with a name, starting tile, and follow mode.
code
#lang survival
 
(define (my-npc)
   (custom-npc
    #:sprite (witch-sprite)
    #:name "Witch"
    #:tile 3
    #:mode 'follow))
 
(survival-game
  #:avatar (custom-avatar)
  #:npc-list (list (my-npc)))

code

(define (my-npc)

(survival-game
  #:avatar (custom-avatar)
  #:npc-list (list (my-npc)))

6.22 Npc Kata (difficulty = 4)

read
Make a survival game with an avatar and two npcs with custom dialog and two other customizations.
code
#lang survival
 
(define (my-npc)
   (custom-npc
    #:name "Francis"
    #:tile 4
    #:dialog (list
     "Greetings!"
     "You better find some food soon...")))
 
(define (another-npc)
   (custom-npc
    #:sprite (witch-sprite)
    #:mode 'pace
    #:dialog (list
     "Now where did I put it..."
     "Have you seen an eye of newt?"
     "Oh, I think I see it!")))
 
(survival-game
  #:avatar (custom-avatar)
  #:npc-list (list (my-npc) (another-npc)))
