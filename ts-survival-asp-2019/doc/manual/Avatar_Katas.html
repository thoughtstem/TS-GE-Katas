2.1 Avatar Kata (difficulty = 1)

read
Make a survival game with a basic avatar.
code
#lang survival
 
(survival-game 
    #:avatar (custom-avatar))

2.2 Avatar Kata (difficulty = 2)

read
Make a survival game with an avatar that looks like a solid yellow star.
code
#lang survival
 
(define (my-avatar) (custom-avatar 
    #:sprite (star 30 'solid 'yellow)))
 
(survival-game 
    #:avatar (my-avatar))

2.3 Avatar Kata (difficulty = 2)

read
Make a survival game with an avatar that is a random-character.
code
#lang survival
 
(define (my-avatar) (custom-avatar 
    #:sprite (random-character-sprite)))
 
(survival-game 
    #:avatar (my-avatar))

2.5 Avatar Kata (difficulty = 2)

read
Make a survival game with an avatar that has a custom animation made in Piskel.
code
#lang survival
 
(define (my-avatar) (custom-avatar 
    #:sprite (sheet->sprite STUDENT-IMAGE-HERE 
    #:columns 4)))
 
(survival-game 
    #:avatar (my-avatar))
