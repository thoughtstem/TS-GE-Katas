3 Enemy Katas

3.1 Bronze Enemy Kata

read
Make a game with an enemy.
code
#lang battle-arena
 
(battle-arena-game
  #:enemy-list (list (custom-enemy)))

3.2 Silver Enemy Kata

read
Make a game with a defined and customized enemy.
code
#lang battle-arena
 
(define (my-enemy)
   (custom-enemy
    #:ai 'easy
    #:health 200
    #:shield 100
    #:amount-in-world 10))
 
(battle-arena-game
  #:enemy-list (list (my-enemy)))

3.3 Gold Enemy Kata

read
Make a game with an enemy that is one of the basic shapes.
code
#lang battle-arena
 
(define (my-enemy)
   (custom-enemy
    #:sprite (star 30 'solid 'gold)
    #:ai 'easy
    #:health 200
    #:shield 100
    #:amount-in-world 10))
 
(battle-arena-game
  #:enemy-list (list (my-enemy)))
