8 Melee Balance Katas

These Katas were considered to be a little too challenging. Before re-using, may want to edit.

8.1 Air Melee Balance Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: Use discussion to help students learn these concepts. They have all played a lot of video games. Help them relate these concepts to games they’ve played – both Fortnite and other games.

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Game Design: Everything involved in making a game. Coding. Art. Balance. Etc.

  • Game Development: Usually, this refers to just the coding aspect of making a game. Not the other parts.

8.2 Bronze Melee Balance Kata (5 minutes)

Review/Introduce: meaning of #lang battle-arena, battle-arena-game, keyword, list.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game that has a light melee weapon.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:weapon-list (list (custom-weapon #:name "Light Melee"
                                     #:dart (custom-dart #:sprite swinging-sword-sprite
                                                         #:damage 50
                                                         #:durability 100
                                                         #:speed  1
                                                         #:range  10
                                                         #:components (every-tick (change-direction-by 15)))
                                     #:rarity 'common)))
   

8.3 Silver Melee Balance Kata (5 minutes)

8.4 Gold Melee Balance Kata (5 minutes)

Review/Introduce: meaning of #lang battle-arena, battle-arena-game, keyword, list.

Extra Dollars for: helping teammates.

Remember!: This kata just combines the previous two. Point this out! It will help students see the pattern.

Students should be able to translate any sentence of this type:
Make a game that has a light repeater and a heavy melee weapon. Balance them relative to each other.
To corresponding code of this type:
  #lang battle-arena
  (define (heavy-dart)
   (custom-dart #:sprite (scale 2 swinging-sword-sprite)
                #:damage 500
                #:durability 200
                #:speed  2
                #:range  10
                #:components (every-tick (change-direction-by 15))))
   
  (define (light-dart)
   (custom-dart #:sprite (scale 2 swinging-sword-sprite)
                #:damage 500
                #:durability 200
                #:speed  2
                #:range  10
                #:components (every-tick (change-direction-by 15))))
   
  (battle-arena-game
  #:weapon-list (list (custom-weapon #:name "Heavy Melee"
                                     #:dart (heavy-dart)
                                     #:rarity 'uncommon)
                      (custom-weapon #:name "Light Melee"
                                     #:dart (light-dart)
                                     #:rarity 'common)))
