13.1 Avatar Katas

13.1.1 Air Avatar Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Player: A human being playing a game

  • Avatar: A player’s represetation in the game

The are NOT the same! (Lots of people in the world don’t know this)

13.1.3 Bronze Avatar Kata (5 minutes)

Review/Introduce: meaning of #lang ts-camp-jam-1, battle-arena-game, avatar, keyword.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game with an avatar.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:avatar (custom-avatar))
   

13.1.4 Silver Avatar Kata (5 minutes)

Review/Introduce: new keyword, the circle function & its parameters.

Extra Dollars for: helping teammates, trying new colors or shapes.

Additional shapes: triangle, square, star, rectangle (needs 2 numbers), ellipse (needs 2 numbers).

Students should be able to translate any sentence of this type:
Make a game with a blue circle avatar.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:avatar (custom-avatar #:sprite (circle 30 'solid 'blue)))
   

13.1.5 Gold Avatar Kata (10 minutes)

Review/Introduce: (my-avatar) can be renamed to anything, the (random-character-sprite) will choose a random sprite each time you run the game.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game with a random-character-sprite as your avatar.
To corresponding code of this type:
  #lang battle-arena
  (define (my-avatar)
   (custom-avatar #:sprite (random-character-sprite)))
   
  (battle-arena-game
  #:avatar (my-avatar))
   

13.1.6 Platinum Avatar Kata (10 minutes)

Review/Introduce: interactions window, sheet->sprite, columns (number of images in the row)

Extra Dollars for: helping teammates.

Tip: The interactions window is the right side of DrRacket where all the code spits out when you run a program.

Students should type the following code into the interactions window:

(random-character-row)

and hit enter to run. Students then can use ctrl + up arrow and enter to rerun this line multiple times.

Once a student finds an avatar they want to use, highlight the image and copy using ctrl + c or right-click, copy. Paste the image (represented by STUDENT_IMAGE_HERE) into the code as seen below.

Students should be able to translate any sentence of this type:
Make a game with a defined custom avatar that has a sprite you chose from the random-character-row list.
To corresponding code of this type:
  #lang battle-arena
  (define (my-avatar)
   (custom-avatar #:sprite (sheet->sprite STUDENT-IMAGE-HERE
                                          #:columns 4)))
   
  (battle-arena-game
  #:avatar (my-avatar))
   

13.2 Size Katas

13.2.1 Bronze Grow Potion Kata (5 minutes)

Review/Introduce: meaning of #lang battle-arena, battle-arena-game, keyword.

Extra Dollars for: helping teammates.

Remember: You can shift the length of time of any Kata to make it the right amount of challenging! Break it Down: Discuss the meaning of the last keyword’s code. What will happen?

Students should be able to translate any sentence of this type:
Make a game that has a potion that makes you big.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:item-list (list (custom-item #:name   "Grow Potion"
                                 #:sprite (make-icon "BIG" 'red 'white)
                                 #:on-use (scale-sprite 2 #:for 100))))
   

13.2.2 Silver Shrink Potion Kata (3 minutes)

To corresponding code of this type:

                                 #:sprite (make-icon "SML" 'blue 'white)
                                 #:on-use (scale-sprite 0.5 #:for 100))))
   

13.2.3 Gold Grow and Shrink Kata (5 minutes)

TIP: For an extra challenge have students add multiple potions and have them complete it in less time.

Students should be able to translate any sentence of this type:
Make a game that has both Grow and Shrink Potions, defined outside battle-arena-game.
To corresponding code of this type:
  #lang battle-arena
  (define (grow-potion)
   (custom-item #:name   "Grow Potion"
                #:sprite (make-icon "BIG" 'red 'white)
                #:on-use (scale-sprite 2 #:for 100)
                #:rarity 'common))
   
  (define (shrink-potion)
   (custom-item #:name     "Shrink Potion"
                #:sprite   (make-icon "SML" 'blue 'white)
                #:on-use   (scale-sprite 0.5 #:for 100)
                #:rarity   'rare))
   
  (battle-arena-game
  #:item-list      (list (grow-potion)
                         (shrink-potion)))

13.6 Example Camp Jam code (w/definitions)

This is an example of how all the components come together during camp-jam into one piece of code:

———————

Note:: This is just an example of some potions and power-ups. Students could have up to 8 or more!

#lang battle-arena
(define (my-avatar)
  (custom-avatar #:sprite (random-character-sprite)))
 
(define (damage-boost)
   (custom-item #:name   "Damage Boost"
                #:sprite (make-icon "DB" 'orangered)
                #:on-use (change-damage-by 1000 #:for 200)
                #:rarity 'epic))
 
 (define (speed-boost)
   (custom-item #:name   "Speed Boost"
                #:sprite (make-icon "SB" 'yellow)
                #:on-use (multiply-speed-by 2 #:for 200)
                #:rarity 'uncommon))
 
(define (shield-potion)
  (custom-item #:name     "Shield Potion"
               #:sprite   (make-icon "SP" 'blue 'white)
               #:on-use   (change-shield-by 50)
               #:rarity   'uncommon
               #:respawn? #t))
 
(define (max-shield)
  (custom-item #:name     "Max Shield Potion"
               #:sprite   (make-icon "MSP" 'blue 'white)
               #:on-use   (set-shield-to 100)
               #:rarity   'epic))
 
(battle-arena-game
 #:avatar      (my-avatar)
 #:item-list   (list (damage-boost)
                     (speed-boost)
                     (shield-potion)
                     (max-shield)))
