7.5 Enemy Katas

Depending on the day; Enemy Katas can be a great "starter" Kata set to run through with all teams before moving on to more challenging katas.

7.5.1 Air Enemy Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: Use discussion to help students learn these concepts. They have all played a lot of video games. Help them relate these concepts to games they’ve played – both Fortnite and other games.

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Enemy Bot: An avatar controlled by artificial intelligence – sometimes used by cheaters, but sometimes used by game designers to test a game.

  • Artificial Intelligence: A computer program that seems to be intelligent, and can sometimes fool human players into thinking it is.

7.5.2 Bronze Enemy Kata (5 minutes)

Review/Introduce: meaning of #lang ts-camp-jam-1, battle-arena-game, keyword, enemy.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game with an enemy.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:enemy-list (list (custom-enemy)))
   

7.5.3 Silver Enemy Kata (5 minutes)

Review/Introduce: all the new keywords, keywords are always optional, (my-enemy) can be anything.

Extra Dollars for: helping teammates.

Hint: AI levels are: ’easy, ’medium, ’hard.

Students should be able to translate any sentence of this type:
Make a game with a defined and customized enemy.
To corresponding code of this type:
  #lang battle-arena
  (define (my-enemy)
   (custom-enemy #:ai              'easy
                 #:health          200
                 #:shield          100
                 #:amount-in-world 10))
   
  (battle-arena-game
  #:enemy-list (list (my-enemy)))
   

7.5.4 Gold Enemy Kata (10 minutes)

Review/Introduce: shapes, the requirements for building a shape.

Extra Dollars for: helping teammates.

Additional shapes: triangle, square, star, rectangle (needs 2 numbers), ellipse (needs 2 numbers).

Students should be able to translate any sentence of this type:
Make a game with an enemy that is one of the basic shapes.
To corresponding code of this type:
  #lang battle-arena
  (define (my-enemy)
   (custom-enemy #:sprite          (star 30 'solid 'gold)
                 #:ai              'easy
                 #:health          200
                 #:shield          100
                 #:amount-in-world 10))
   
  (battle-arena-game
  #:enemy-list (list (my-enemy)))
   

7.5.5 Platinum Enemy Kata (10 minutes)

Review/Introduce: interactions window, sheet->sprite, columns (number of images in the row)

Extra Dollars for: helping teammates.

Tip: The interactions window is the right side of DrRacket where all the code spits out when you run a program.

Students should type the following code into the interactions window:

(random-character-row)

and hit enter to run. Students then can use ctrl + up arrow and enter to rerun this line multiple times.

Once a student finds a sprite they want to use for their enemy, highlight the image and copy using ctrl + c or right-click, copy. Paste the image (represented by STUDENT_IMAGE_HERE) into the code as seen below.

Students should be able to translate any sentence of this type:
Make a game with a defined custom enemy that has a sprite you chose from the random-character-row list.
To corresponding code of this type:
  #lang battle-arena
  (define (my-enemy)
   (custom-enemy #:sprite          (sheet->sprite STUDENT-IMAGE-HERE
                                                  #:columns 4)
                 #:ai              'easy
                 #:health          200
                 #:shield          100
                 #:amount-in-world 10))
   
  (battle-arena-game
  #:enemy-list (list (my-enemy)))
   

5.2 Enemy Weapon Katas

5.2.1 Bronze Enemy Weapon Kata (5 minutes)

Review/Introduce: meaning of #lang ts-camp-jam-1, battle-arena-game, keyword, list, dart.

Extra Dollars for: helping teammates.

Tip: In addition to sword you can use any of these: paint, spear, lava-builder, spike-mine-builder, or spear-tower-builder.

Students should be able to translate any sentence of this type:
Make a game that has a an enemy and give it a custom weapon.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:enemy-list (list (custom-enemy
                      #:weapon (custom-weapon
                                #:dart (sword)))))
   

5.2.2 Silver Enemy Weapon Kata (5 minutes)

Review/Introduce: meaning of damage, size, multiple keywords.

Extra Dollars for: helping teammates.

Tip: Remember, all the keywords (denoted by #: ) are always optional.

Students should be able to translate any sentence of this type:
Make a game that has an enemy, give it a custom weapon and customize it.
To corresponding code of this type:
  #lang battle-arena
  (define (my-weapon)
   (custom-weapon
    #:name      "Sword"
    #:dart      (sword #:damage 40)))
   
  (battle-arena-game
  #:enemy-list (list (custom-enemy
                      #:weapon (my-weapon))))
   

5.2.3 Gold Enemy Weapon Kata (5 minutes)

Review/Introduce: meaning of sprite, range.

Extra Dollars for: helping teammates, creativity.

Tip: Remember, all the keywords (denoted by #: ) are always optional.

Students should be able to translate any sentence of this type:
Make a game that has am enemy, give it a custom weapon, and customize the dart.
To corresponding code of this type:
  #lang battle-arena
  (define (my-dart)
   (custom-dart
    #:damage 5
    #:speed  1
    #:range  200))
   
  (define (my-weapon)
   (custom-weapon
    #:name      "Repeator"
    #:dart      (my-dart)))
   
  (battle-arena-game
  #:enemy-list (list (custom-enemy
                      #:weapon (my-weapon))))
   

5.3 Repeater Armor Katas

5.3.1 Bronze Repeater Armor Kata (5 minutes)

Review/Introduce: meaning of #lang ts-camp-jam-1, battle-arena-game, keyword, list, what make-icon does.

Extra Dollars for: helping teammates.

Tip: This isn’t going to work to to protect your avatar yet! That’s in the next kata.

Students should be able to translate any sentence of this type:
Make a game that has armor called Repeater Armor.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:item-list (list (custom-armor #:name   "Repeater Armor"
                                  #:sprite (make-icon "RA"))))
   

5.3.2 Silver Repeater Armor Kata (5 minutes)

Review/Introduce: figure out why #:change-damage has (divide by).

Extra Dollars for: helping teammates.

Tip: The name after the #:protects-from keyword must match the name of the weapon held by the enemy.

Rarities: All rarity levels are ’common, ’uncommon, ’rare, ’epic, and ’legendary.

Students should be able to translate any sentence of this type:
Make a game that has working Repeater Armor.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:item-list (list (custom-armor #:name          "Repeater Armor"
                                  #:sprite        (make-icon "RA")
                                  #:protects-from "Repeater"
                                  #:change-damage (divide-by 2)
                                  #:rarity        'rare)))
   

5.3.3 Gold Repeater Armor Kata (5 minutes)

Review/Introduce: meaning of damage.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game that has 10 enemies with a repeater and armor for your avatar that will protect against that repeater!
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:enemy-list (list (custom-enemy #:amount-in-world 10
                                   #:weapon          (custom-weapon
                                                      #:name "Repeater")))
  #:item-list (list (custom-armor #:name          "Repeater Armor"
                                  #:sprite        (make-icon "RA")
                                  #:protects-from "Repeater"
                                  #:change-damage (divide-by 2)
                                  #:rarity        'rare)))

8.1 Avatar Katas

Depending on the day; Avatar Katas can be a great "starter" Kata set to run through with all teams before moving on to more challenging katas.

8.1.1 Air Avatar Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Player: A human being playing a game

  • Avatar: A player’s represetation in the game

The are NOT the same! (Lots of people in the world don’t know this)

5.6 Example Camp Jam code (w/definitions)

This is an example of how all the components come together during camp-jam into one piece of code:

———————

#lang battle-arena
(define (my-avatar)
  (custom-avatar #:sprite (random-character-sprite)))
 
(define (badguy-weapon)
  (custom-weapon #:name        "Sword"
                 #:dart        (sword #:damage 40)))
 
(define (badguy)
  (custom-enemy #:ai              'easy
                #:health          200
                #:shield          100
                #:amount-in-world 10
                #:weapon          (badguy-weapon)))
 
(define (my-armor)
  (custom-armor #:name          "Sword Armor"
                #:sprite        (make-icon "SA")
                #:protects-from "Sword"
                #:change-damage (subtract-by 30)
                #:rarity        'rare))
 
(battle-arena-game #:avatar     (my-avatar)
                   #:enemy-list (list (badguy))
                   #:item-list  (list (my-armor)))
