5 Sword Katas

5.1 Air Sword Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: Use discussion to help students learn these concepts. They have all played a lot of video games. Help them relate these concepts to games they’ve played – both Fortnite and other games.

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Combat system: A system in some (but not all) games. It usually involves weapons, damage, healing, and death.

  • Weapon: An item that avatars can use to damage enemies

  • Damage: Reducing one or more of an avatar’s stats (like health or shields).

  • Death: When an avatar is removed from a game or must respawn due to complete loss of health.

5.2 Bronze Sword Kata (5 minutes)

Review/Introduce: meaning of #lang battle-arena, battle-arena-game, list, keywords.

Extra Dollars for: helping teammates.

Tip: "Sword" can be anything, like "Amazing Sword of Awesome"

Students should be able to translate any sentence of this type:
Make a game that has a sword.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:weapon-list (list (custom-weapon #:name    "Sword"
                                     #:sprite  SWORD-ICON
                                     #:dart    (sword))))
   

5.3 Silver Sword Kata (5 minutes)

Review/Introduce: new keyword, customizable function names (my-weapon-2).

Extra Dollars for: helping teammates.

Tip: Rarity indicates how many there will be in the world. All options are: ’common, ’uncommon, ’rare, ’epic, ’lengendary.

Students should be able to translate any sentence of this type:
Make a game with your own defined sword.
To corresponding code of this type:
  #lang battle-arena
  (define (my-weapon-2)
   (custom-weapon #:name    "Sword"
                  #:sprite  SWORD-ICON
                  #:dart    (sword)
                  #:rarity  'rare))
   
  (battle-arena-game
  #:weapon-list (list (my-weapon-2)))
   
Bonus Code: Have them add additional keywords to (custom-weapon) to earn an extra dollar!

  #lang battle-arena
  (define (my-weapon-2)
   (custom-weapon #:name              "Sword"
                  #:sprite            SWORD-ICON
                  #:dart              (sword)
                  #:rarity            'rare
                  #:mouse-fire-button 'left
                  #:rapid-fire?        #f))
   
  (battle-arena-game
  #:weapon-list (list (my-weapon-2)))
   

5.4 Gold Sword Kata (10 minutes)

Review/Introduce: function names can be anything, draw simple – nothing is saved for later.

Extra Dollars for: helping teammates, creativity.

Tip: Give students the piskel cheat sheet.

Students should be able to translate any sentence of this type:
Make a game with a fully customized sword.
To corresponding code of this type:
  #lang battle-arena
  (define (my-sword)
   (sword #:damage     50
          #:durability 20))
   
  (define (my-weapon-2)
   (custom-weapon #:name   "Sword"
                  #:sprite STUDENT-IMAGE-HERE
                  #:dart   (my-sword)))
   
  (battle-arena-game
  #:weapon-list (list (my-weapon-2)))
   

5.5 Platinum Sword Kata (10 minutes)

Review/Introduce: what do the new keywords mean?

Extra Dollars for: helping teammates, creativity.

Students should be able to translate any sentence of this type:
Make a game that has a customizable sword in it where you can modify its speed, range, and sprite.
To corresponding code of this type:
  #lang battle-arena
  (define (my-sword)
   (sword #:damage     50
          #:durability 20
          #:sprite     STUDENT-IMAGE-HERE
          #:speed      0
          #:range      10))
   
  (define (my-weapon-2)
   (custom-weapon #:name   "Sword"
                  #:sprite STUDENT-IMAGE-HERE
                  #:dart   (my-sword)))
   
  (battle-arena-game
  #:weapon-list (list (my-weapon-2)))
