6 Paint Thrower Katas

6.1 Air Paint Thrower Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: Use discussion to help students learn these concepts. They have all played a lot of video games. Help them relate these concepts to games they’ve played – both Fortnite and other games.

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Melee weapon: A kind of weapon that is only effective at short range.

  • Ranged weapon: A kind of weapon that is effective at long range.

6.3 Bronze Paint Thrower Kata (5 minutes)

Review/Introduce: meaning of #lang battle-arena, battle-arena-game, list, keywords

Extra Dollars for: helping teammates.

Tip: Remember, keyword arguments (the parts of the code that start with "#:") are optional!)

Students should be able to translate any sentence of this type:
Make a game that has a paint thrower in it.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:weapon-list (list (custom-weapon #:name   "Paint Thrower"
                                     #:sprite PAINT-THROWER-ICON
                                     #:dart   (paint))))
   

6.4 Silver Paint Thrower Kata (5 minutes)

Review/Introduce: (my-weapon-3) could be anything; you could rename it to (paint-spitter)

Extra Dollars for: bonus code, helping teammates.

Hint: rarity levels are ’common, ’uncommon, ’rare, ’epic, ’legendary. Rarity is related to how many will spawn in the world!

Students should be able to translate any sentence of this type:
Make a game with your own defined paint thrower with customizable rarity.
To corresponding code of this type:
  #lang battle-arena
  (define (my-weapon-3)
   (custom-weapon #:name   "Paint Thrower"
                  #:sprite PAINT-THROWER-ICON
                  #:dart   (paint)
                  #:rarity 'epic))
   
  (battle-arena-game
  #:weapon-list (list (my-weapon-3)))
   

Bonus Code: Have them add additional keywords to (custom-weapon) to earn an extra dollar!

  #lang battle-arena
  (define (my-weapon-3)
   (custom-weapon #:name              "Paint Thrower"
                  #:sprite            PAINT-THROWER-ICON
                  #:dart              (paint)
                  #:rarity            'epic
                  #:fire-rate         30
                  #:mouse-fire-button 'left))
   
  (battle-arena-game
  #:weapon-list (list (my-weapon-3)))
   

6.5 Gold Paint Thrower Kata (10 minutes)

Review/Introduce: (my-paint) can be renamed to anything you’d like!

Extra Dollars for: helping teammates, creativity.

Tip: Use the piskel cheat sheet! Have them draw fast & simple - they’re not going to keep it!

Students should be able to translate any sentence of this type:
Make a game that has a customizable paint thrower in it where you can modify the damage and durability of the paint.
To corresponding code of this type:
  #lang battle-arena
  (define (my-paint)
   (paint #:damage     10
          #:durability 20))
   
  (define (my-weapon-3)
   (custom-weapon #:name   "Paint Thrower"
                  #:sprite STUDENT-IMAGE-HERE
                  #:dart   (my-paint)
                  #:rarity 'epic))
   
  (battle-arena-game
  #:weapon-list (list (my-weapon-3)))
   

6.6 Platinum Paint Thrower Kata (10 minutes)

Review/Introduce: what do the new keywords mean?

Extra Dollars for: helping teammates, creativity.

Students should be able to translate any sentence of this type:
Make a game that has a customizable paint thrower in it where you can modify the speed, range, and sprite of the paint.
To corresponding code of this type:
  #lang battle-arena
  (define (my-paint)
   (paint #:damage     10
          #:durability 20
          #:sprite     paint-sprite
          #:speed      3
          #:range      15))
   
  (define (my-weapon-3)
   (custom-weapon #:name   "Paint Thrower"
                  #:sprite STUDENT-IMAGE-HERE
                  #:dart   (my-paint)
                  #:rarity 'epic))
   
  (battle-arena-game
