2 Melee Battle Mode Katas

2.1 Spear Katas

2.1.1 Air Spear Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: Use discussion to help students learn these concepts. They have all played a lot of video games. Help them relate these concepts to games they’ve played – both Fortnite and other games.

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Asset: The fundamental building material of a game. Can be: art, music, sound effects, code, etc. Game designers use programming to combine assets into a game.

  • Funny Asset: Any asset that is intended to make players laugh.

One of the things that makes Fortnite famous are its funny assets. What’s your favorite!

2.1.2 Bronze Spear Kata (5 minutes)

Review/Introduce: meaning of #lang battle-arena, battle-arena-game, keyword, list.

Extra Dollars for: helping teammates.

Hint: "Spear" can be anything, like "Amaze-spear".

Students should be able to translate any sentence of this type:
Make a game that has a spear.
To corresponding code of this type:
  #lang battle-arena
  
  (battle-arena-game
  #:weapon-list (list (custom-weapon #:name   "Spear"
                                     #:sprite SPEAR-ICON
                                     #:dart (spear))))
   

2.1.3 Silver Spear Kata (5 minutes)

Review/Introduce: new keyword, customizable function names (my-weapon-1).

Extra Dollars for: bonus code, helping teammates.

Tip: Rarity indicates how many there will be in the world. All options are: ’common, ’uncommon, ’rare, ’epic, ’lengendary.

Students should be able to translate any sentence of this type:
Make a game with your own defined spear.
To corresponding code of this type:
  #lang battle-arena
  
  (define (my-weapon-1)
  
   (custom-weapon #:name   "Spear"
                  #:sprite SPEAR-ICON
                  #:dart (spear)
                  #:rarity 'common)) (battle-arena-game
  #:weapon-list (list (my-weapon-1)))
   

Bonus Code: Have them add additional keywords to (custom-weapon) to earn an extra dollar!

  #lang battle-arena
  
  (define (my-weapon-1)
  
   (custom-weapon #:name              "Spear"
                  #:sprite          SPEAR-ICON
                  #:dart            (spear)
                  #:rarity        'common
                  #:mouse-fire-button 'left
                  #:rapid-fire?       #f)) (battle-arena-game
  #:weapon-list (list (my-weapon-1)))
   

2.1.4 Gold Spear Kata (10 minutes)

Review/Introduce: (my-weapon-1) and (my-spear) can be renamed to anything you’d like!

Extra Dollars for: helping teammates, creativity.

Tip: Use the piskel cheat sheet! Have them draw fast & simple - they’re not going to keep it!

Students should be able to translate any sentence of this type:
Make a game that has a fully customizable spear in it where you can modify its damage and durability.
To corresponding code of this type:
  #lang battle-arena
  
  (define (my-spear)
  
   (spear #:damage     25
          #:durability 10)) (define (my-weapon-1) (custom-weapon #:name   "Spear"
                  #:sprite STUDENT-IMAGE-HERE
                  #:dart (my-spear)
                  #:rarity 'common)) (battle-arena-game
  #:weapon-list (list (my-weapon-1)))
   

2.1.5 Platinum Spear Kata (10 minutes)

Review/Introduce: what do the new keywords mean?

Extra Dollars for: helping teammates, creativity.

Students should be able to translate any sentence of this type:
Make a game that has a fully customizable spear in it where you can modify its speed, range, and sprite.
To corresponding code of this type:

  (define (my-spear)
  
   (spear #:damage     25
          #:durability 10
          #:sprite  spear-sprite
          #:speed      5
          #:range      20))
  (define (my-weapon-1) (custom-weapon #:name   "Spear"
                  #:sprite STUDENT-IMAGE-HERE
                  #:dart   (my-spear)
                #:rarity 'common)) (battle-arena-game
  
  #:weapon-list (list (my-weapon-1)))
   

2.2 Sword Katas

2.2.1 Air Sword Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: Use discussion to help students learn these concepts. They have all played a lot of video games. Help them relate these concepts to games they’ve played – both Fortnite and other games.

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Combat system: A system in some (but not all) games. It usually involves weapons, damage, healing, and death.

  • Weapon: An item that avatars can use to damage enemies

  • Damage: Reducing one or more of an avatar’s stats (like health or shields).

  • Death: When an avatar is removed from a game or must respawn due to complete loss of health.

5.5 Avatar Katas

Depending on the day; Avatar Katas can be a great "starter" Kata set to run through with all teams before moving on to more challenging katas.

5.5.1 Air Avatar Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Player: A human being playing a game

  • Avatar: A player’s represetation in the game

The are NOT the same! (Lots of people in the world don’t know this)

Review/Introduce: core values

Extra Dollars for: lively discussion

Format: It’s probably not important to officially time students on this kata. It’s also okay to encourage lively discussion. Some of the pictures are funny, and that’s okay.

Students should be able to identify each of the following as either avatars or players:

5.5.3 Bronze Avatar Kata (5 minutes)

Review/Introduce: meaning of #lang ts-camp-jam-1, battle-arena-game, avatar, keyword.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game with an avatar.

5.5.4 Silver Avatar Kata (5 minutes)

Review/Introduce: new keyword, the circle function & its parameters.

Extra Dollars for: helping teammates, trying new colors or shapes.

Additional shapes: triangle, square, star, rectangle (needs 2 numbers), ellipse (needs 2 numbers).

Students should be able to translate any sentence of this type:
Make a game with a blue circle avatar.
To corresponding code of this type:
  #lang battle-arena
  
  (battle-arena-game
  #:avatar (custom-avatar #:sprite (circle 30 'solid 'blue)))
   

5.5.5 Gold Avatar Kata (10 minutes)

Review/Introduce: (my-avatar) can be renamed to anything, the (random-character-sprite) will choose a random sprite each time you run the game.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game with a random-character-sprite as your avatar.
To corresponding code of this type:
  #lang battle-arena
  
  (define (my-avatar)
  
   (custom-avatar #:sprite (random-character-sprite)))
   (battle-arena-game
  #:avatar (my-avatar))
   

5.5.6 Platinum Avatar Kata (10 minutes)

Review/Introduce: interactions window, sheet->sprite, columns (number of images in the row)

Extra Dollars for: helping teammates.

Tip: The interactions window is the right side of DrRacket where all the code spits out when you run a program.

Students should type the following code into the interactions window:

(random-character-row)

and hit enter to run. Students then can use ctrl + up arrow and enter to rerun this line multiple times.

Once a student finds an avatar they want to use, highlight the image and copy using ctrl + c or right-click, copy. Paste the image (represented by STUDENT_IMAGE_HERE) into the code as seen below.

Students should be able to translate any sentence of this type:
Make a game with a defined custom avatar that has a sprite you chose from the random-character-row list.
To corresponding code of this type:
  #lang battle-arena
  
  (define (my-avatar)
  
   (custom-avatar #:sprite (sheet->sprite STUDENT-IMAGE-HERE
                                          #:columns 4))) (battle-arena-game
  #:avatar (my-avatar))
   

5.1 Enemy Katas

Depending on the day; Enemy Katas can be a great "starter" Kata set to run through with all teams before moving on to more challenging katas.

5.1.1 Air Enemy Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: Use discussion to help students learn these concepts. They have all played a lot of video games. Help them relate these concepts to games they’ve played – both Fortnite and other games.

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Enemy Bot: An avatar controlled by artificial intelligence – sometimes used by cheaters, but sometimes used by game designers to test a game.

  • Artificial Intelligence: A computer program that seems to be intelligent, and can sometimes fool human players into thinking it is.

5.1.3 Silver Enemy Kata (5 minutes)
