4 Defensive Base Katas

4.1 Lava Katas

4.1.1 Bronze Lava Pit Kata (5 minutes)

Review/Introduce: meaning of #lang ts-camp-jam-1, battle-arena-game, keyword, list, dart.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game that has a custom weapon that deploys a lava pit.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:weapon-list (list (custom-weapon
                       #:name "Lava Pit"
                       #:sprite (make-icon "LP")
                       #:dart (lava-builder))))
   

4.1.2 Silver Lava Pit Kata (5 minutes)

Review/Introduce: meaning of damage, size, multiple keywords.

Extra Dollars for: helping teammates.

4.1.3 Gold Lava Pit Kata (5 minutes)

Review/Introduce: meaning of sprite, range.

Extra Dollars for: helping teammates, creativity.

Students should be able to translate any sentence of this type:
Make a game that has a custom weapon that deploys a lava pit with custom damage, size, sprite, and range.
To corresponding code of this type:

6.4 Avatar Katas

Depending on the day; Avatar Katas can be a great "starter" Kata set to run through with all teams before moving on to more challenging katas.

6.4.1 Air Avatar Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Player: A human being playing a game

  • Avatar: A player’s represetation in the game

The are NOT the same! (Lots of people in the world don’t know this)

Review/Introduce: meaning of #lang ts-camp-jam-1, battle-arena-game, avatar, keyword.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:

6.4.4 Silver Avatar Kata (5 minutes)

Review/Introduce: new keyword, the circle function & its parameters.

Extra Dollars for: helping teammates, trying new colors or shapes.

Additional shapes: triangle, square, star, rectangle (needs 2 numbers), ellipse (needs 2 numbers).

Students should be able to translate any sentence of this type:
Make a game with a blue circle avatar.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:avatar (custom-avatar #:sprite (circle 30 'solid 'blue)))
   

6.4.5 Gold Avatar Kata (10 minutes)

Review/Introduce: (my-avatar) can be renamed to anything, the (random-character-sprite) will choose a random sprite each time you run the game.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game with a random-character-sprite as your avatar.
To corresponding code of this type:
  #lang battle-arena
  (define (my-avatar)
   (custom-avatar #:sprite (random-character-sprite)))
   
  (battle-arena-game
  #:avatar (my-avatar))
   

6.4.6 Platinum Avatar Kata (10 minutes)

Review/Introduce: interactions window, sheet->sprite, columns (number of images in the row)

Extra Dollars for: helping teammates.

Tip: The interactions window is the right side of DrRacket where all the code spits out when you run a program.

Students should type the following code into the interactions window:

(random-character-row)

and hit enter to run. Students then can use ctrl + up arrow and enter to rerun this line multiple times.

Once a student finds an avatar they want to use, highlight the image and copy using ctrl + c or right-click, copy. Paste the image (represented by STUDENT_IMAGE_HERE) into the code as seen below.

Students should be able to translate any sentence of this type:
Make a game with a defined custom avatar that has a sprite you chose from the random-character-row list.
To corresponding code of this type:
  #lang battle-arena
  (define (my-avatar)
   (custom-avatar #:sprite (sheet->sprite STUDENT-IMAGE-HERE
                                          #:columns 4)))
   
  (battle-arena-game
  #:avatar (my-avatar))
   

6.5 Enemy Katas

Depending on the day; Enemy Katas can be a great "starter" Kata set to run through with all teams before moving on to more challenging katas.

6.5.1 Air Enemy Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: Use discussion to help students learn these concepts. They have all played a lot of video games. Help them relate these concepts to games they’ve played – both Fortnite and other games.

Game design vocab!

Students should be able to recite the following definitions from memory:

  • Enemy Bot: An avatar controlled by artificial intelligence – sometimes used by cheaters, but sometimes used by game designers to test a game.

  • Artificial Intelligence: A computer program that seems to be intelligent, and can sometimes fool human players into thinking it is.

6.5.2 Bronze Enemy Kata (5 minutes)

Review/Introduce: meaning of #lang ts-camp-jam-1, battle-arena-game, keyword, enemy.

Extra Dollars for: helping teammates.

Students should be able to translate any sentence of this type:
Make a game with an enemy.
To corresponding code of this type:
  #lang battle-arena
  (battle-arena-game
  #:enemy-list (list (custom-enemy)))
   

6.5.3 Silver Enemy Kata (5 minutes)

Review/Introduce: all the new keywords, keywords are always optional, (my-enemy) can be anything.

Extra Dollars for: helping teammates.

Hint: AI levels are: ’easy, ’medium, ’hard.

Students should be able to translate any sentence of this type:
Make a game with a defined and customized enemy.
To corresponding code of this type:
  #lang battle-arena
  (define (my-enemy)
   (custom-enemy #:ai              'easy
                 #:health          200
                 #:shield          100
                 #:amount-in-world 10))
   
  (battle-arena-game
  #:enemy-list (list (my-enemy)))

Extra Dollars for: helping teammates.

Additional shapes: triangle, square, star, rectangle (needs 2 numbers), ellipse (needs 2 numbers).

Students should be able to translate any sentence of this type:
Make a game with an enemy that is one of the basic shapes.
To corresponding code of this type:
