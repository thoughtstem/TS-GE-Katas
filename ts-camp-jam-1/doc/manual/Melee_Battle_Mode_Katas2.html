2 Melee Battle Mode Katas2

2.1 Bronze Spear Kata

When given this English phrase: Make a game that has a spear.
You can write this battle-arena code:
#lang battle-arena
(battle-arena-game
#:weapon-list (list (custom-weapon #:name   "Spear"
                                   #:sprite SPEAR-ICON
                                   #:dart   (spear))))
 

2.2 Silver Spear Kata

When given this English phrase: Make a game with your own defined spear.
You can write this battle-arena code:
#lang battle-arena
(define (my-weapon-1)
 (custom-weapon #:name   "Spear"
                #:sprite SPEAR-ICON
                #:dart   (spear)
                #:rarity 'common))
 
(battle-arena-game
#:weapon-list (list (my-weapon-1)))
 

2.3 Gold Spear Kata

When given this English phrase: Make a game that has a fully customizable spear in it where you can modify its damage and durability.

 (spear #:damage     25
        #:durability 10))
 
(define (my-weapon-1)

#:weapon-list (list (my-weapon-1)))
