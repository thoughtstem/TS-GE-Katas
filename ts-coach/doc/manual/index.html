Unnamed Kata Collection

This is not a Kata Collection.

This is a web page that displays information that came from a kata collection. Neither the web page, nor the Scribble file that created it are kata collections.

The collection is in katas.rkt. If you are editing kata data, do so there (or in whatever rkt files are appropriate).

If you are creating a custom way of displaying a kata collection, THAT’s what you would use Scribble for.
