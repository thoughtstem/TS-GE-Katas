6 Sky Katas

6.1 Sky Kata (sky-1)

read
Make a survival game with an avatar and a slow changing sky.
code
#lang survival
 
(survival-game 
    #:avatar (custom-avatar) 
    #:sky (custom-sky 
    #:length-of-day 5000))

6.2 Sky Kata (sky-2)

read
Make a survival game with an avatar and a fast changing sky with max-darkness.
code
#lang survival
 
(survival-game 
    #:avatar (custom-avatar) 
    #:sky (custom-sky 
    #:length-of-day 500 
    #:max-darkness 255))

6.3 Sky Kata (sky-3)

read
Make a survival game with an avatar and a darkmagenta night-sky with 150 darkness.
code
#lang survival
 
(survival-game 
    #:avatar (custom-avatar) 
    #:sky (custom-sky 
    #:night-sky-color 'darkmagenta 
    #:max-darkness 150))

6.4 Sky Kata (sky-4)

read
Make a survival game with an avatar, a very short night-time, and an enemy that only comes out at night.
code
#lang survival
 
(survival-game 
    #:avatar (custom-avatar) 
    #:enemy-list (list (custom-enemy 
    #:amount-in-world 20 
    #:night-only? #t)) 
    #:sky (custom-sky 
    #:length-of-day 2400 
    #:start-of-daytime 200 
    #:end-of-daytime 2200))
