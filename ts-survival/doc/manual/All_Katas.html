11.2 Avatar Kata (avatar-2)

read
Make a survival game with an avatar with a sprite picked from the assets library.
code
#lang survival
 
(define (my-avatar)
   (custom-avatar #:sprite wizard-sprite))
 
(survival-game #:avatar (my-avatar))

11.3 Avatar Kata (avatar-3)

Tip: The default speed is 10.

read
Make a survival game with an avatar with a custom sprite and speed of 20.
code
#lang survival
 
(define (my-avatar)
   (custom-avatar
    #:sprite pirate-sprite
    #:speed 20))
 
(survival-game #:avatar (my-avatar))

11.4 Avatar Kata (avatar-4)

Tip: The default key-mode is ’arrow-keys.

read
Make a survival game with an avatar with a custom sprite, speed and WASD key-mode.
code
#lang survival
 
(define (my-avatar)
   (custom-avatar
    #:sprite monk-sprite
    #:speed 20
    #:key-mode 'wasd))
 
(survival-game #:avatar (my-avatar))

11.13 Crafter Kata (crafter-2)

Tip: By adding a recipe, the crafter now requires an item in the avatar’s backpack to build something.

read
Make a survival game with an avatar, carrots, and a crafter that makes carrot stew from carrots.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:food-list (list (carrot #:amount-in-world 10))
  #:crafter-list (list
   (custom-crafter
    #:recipe-list (list carrot-stew-recipe))))

11.14 Crafter Kata (crafter-3)

read
Make a survival game with an avatar, fish, and a crafter that makes fish stew that heals you by 50.
code
#lang survival
 
(define (fish-stew)
   (custom-food
    #:name "Fish Stew"
    #:sprite fish-stew-sprite
    #:respawn? #f
    #:heals-by 50))
 
(define fish-stew-recipe
   (recipe
    #:product (fish-stew)
    #:build-time 40
    #:ingredients (list "Fish")))
 
(survival-game
  #:avatar (custom-avatar)
  #:food-list (list (fish #:amount-in-world 10))
  #:crafter-list (list
   (custom-crafter
    #:sprite cauldron-sprite
    #:recipe-list (list fish-stew-recipe))))

11.15 Crafter Kata (crafter-4)

read
Make a survival game with an avatar, carrots, fish, and a crafter on tile 2 (posn 200 200) that makes both carrot-stew and fish-stew.
code
#lang survival
 
(define (fish-stew)
   (custom-food
    #:name "Fish Stew"
    #:sprite fish-stew-sprite
    #:respawn? #f
    #:heals-by 50))
 
(define fish-stew-recipe
   (recipe
    #:product (fish-stew)
    #:build-time 40
    #:ingredients (list "Fish")))
 
(define (my-cauldron)
   (custom-crafter
    #:sprite cauldron-sprite
    #:position (posn 200 200)
    #:tile 2
    #:recipe-list (list carrot-stew-recipe fish-stew-recipe)))
 
(survival-game
  #:avatar (custom-avatar)
  #:food-list (list
   (carrot #:amount-in-world 10)
   (fish #:amount-in-world 10))
  #:crafter-list (list (my-cauldron)))

11.17 Enemy Kata (enemy-2)

Tip: Without the curry, the game will have 10 of the same enemy. With curry, the game will have 10 different enemies.

read
Make a survival game with an avatar and 10 random enemies.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:enemy-list (list
   (curry custom-enemy #:amount-in-world 10)))
