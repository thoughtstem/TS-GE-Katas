9.1 Npc Kata (npc-1)

read
Make a survival game with an avatar and a basic npc.
code
#lang survival
 
(survival-game 
    #:avatar (custom-avatar) 
    #:npc-list (list (custom-npc)))

9.2 Npc Kata (npc-2)

read
Make a survival game with an avatar and a witch npc with a name.
code
#lang survival
 
(define (my-npc) (custom-npc 
    #:sprite (witch-sprite) 
    #:name "Witch"))
 
(survival-game 
    #:avatar (custom-avatar) 
    #:npc-list (list (my-npc)))

9.3 Npc Kata (npc-3)

read
Make a survival game with an avatar and a witch npc with a name, starting tile, and follow mode.
code
#lang survival
 
(define (my-npc) (custom-npc 
    #:sprite (witch-sprite) 
    #:name "Witch" 
    #:tile 3 
    #:mode 'follow))
 
(survival-game 
    #:avatar (custom-avatar) 
    #:npc-list (list (my-npc)))

9.5 Npc Kata (npc-5)

read
Make a survival game with an avatar and two npcs with custom dialog and two other customizations.
code
#lang survival
 
(define (my-npc) (custom-npc 
    #:name "Francis" 
    #:tile 4 
    #:dialog (list "Greetings!" "You better find some food soon...")))
 
(define (another-npc) (custom-npc 
    #:sprite (witch-sprite) 
    #:mode 'pace 
    #:dialog (list "Now where did I put it..." "Have you seen an eye of newt?" "Oh, I think I see it!")))
 
(survival-game 
    #:avatar (custom-avatar) 
    #:npc-list (list (my-npc) (another-npc)))
