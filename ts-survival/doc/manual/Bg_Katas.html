9 Bg Katas

9.1 Bg Kata (bg-1)

read
Make a survival game with an avatar and a basic custom-bg.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:bg (custom-bg))

9.2 Bg Kata (bg-2)

Tip: See other bg-image options in the assets library.

read
Make a survival game with an avatar and a desert custom-bg.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:bg (custom-bg #:image DESERT-BG))

9.3 Bg Kata (bg-3)

Tip: start-tile can take a number between 0 (the default) and 9.

read
Make a survival game with an avatar and a lava custom-bg where the avatar starts on tile 4.
code
#lang survival
 
(define (my-bg)
   (custom-bg #:image LAVA-BG #:start-tile 4))
 
(survival-game
  #:avatar (custom-avatar)
  #:bg (my-bg))
