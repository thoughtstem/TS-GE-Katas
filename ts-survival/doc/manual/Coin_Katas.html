4.3 Coin Kata (coin-3)

read
Make survival game with an avatar and 20 silver coins worth 500
code
#lang survival
 
(define (my-coin)
   (custom-coin
    #:sprite
    silver-coin-sprite
    #:name
    "Silver Coin"
    #:value
    500
    #:amount-in-world
    20))
 
(survival-game
  #:avatar
  (custom-avatar)
  #:coin-list
  (list (my-coin)))

4.4 Coin Kata (coin-4)

read
Make survival game with an avatar, a silver coin, and a single epic gold coin that never respawns and is worth 1000
code
#lang survival
 
(define (my-coin)
   (custom-coin
    #:sprite
    silver-coin-sprite
    #:name
    "Silver Coin"))
 
(define (my-special-coin)
   (custom-coin
    #:sprite
    gold-coin-sprite
    #:name
    "Gold Coin"
    #:value
    1000
    #:amount-in-world
    1
    #:respawn?
    #f))
 
(survival-game
  #:avatar
  (custom-avatar)
  #:coin-list
  (list (my-coin) (my-special-coin)))
