4 Enemy Katas

4.1 Enemy Kata (enemy-1)

read
Make a survival game with an avatar and an enemy.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:enemy-list (list (custom-enemy)))

4.2 Enemy Kata (enemy-5)

read
Make a survival game with an avatar and 5 hard bat enemies that spit acid with 50 damage.
code
#lang survival
 
(define (hard-enemy)
   (custom-enemy
    #:ai 'hard
    #:sprite snake-sprite
    #:amount-in-world 5
    #:weapon (acid-spitter #:damage 50)))
 
(survival-game
  #:avatar (custom-avatar)
  #:enemy-list (list (hard-enemy)))

4.3 Enemy Kata (enemy-2)

Tip: Without the curry, the game will have 10 of the same enemy. With curry, the game will have 10 different enemies.

read
Make a survival game with an avatar and 10 random enemies.
code
#lang survival
 
(survival-game
  #:avatar (custom-avatar)
  #:enemy-list (list
   (curry custom-enemy #:amount-in-world 10)))

4.4 Enemy Kata (enemy-3)

Tip: The options for #:ai are ’easy, ’medium and ’hard.

read
Make a survival game with an avatar and 5 bat enemies with medium AI.
code
#lang survival
 
(define (my-enemy)
   (custom-enemy
    #:ai 'medium
    #:sprite bat-sprite
    #:amount-in-world 5))
 
(survival-game
  #:avatar (custom-avatar)
  #:enemy-list (list (my-enemy)))

4.5 Enemy Kata (enemy-4)

read
Make a survival game with an avatar, 5 easy slime enemies, and 5 medium bat enemies that only come out at night.
code
#lang survival
 
(define (easy-enemy)
   (custom-enemy
    #:ai 'easy
    #:sprite slime-sprite
    #:amount-in-world 5))
 
(define (medium-enemy)
   (custom-enemy
    #:ai 'medium
    #:sprite bat-sprite
    #:amount-in-world 5
    #:night-only? #t))
 
(survival-game
  #:avatar (custom-avatar)
  #:enemy-list (list (easy-enemy) (medium-enemy)))
