2 One Day Camp Materials

All the materials needed to run a successful full-day, one day camp!

2.1 Name Tags/Mini Health Bars

# = 1 per student

Each student receives one of the following at the very beginning of the day.

Write their name (or have the student write it!) on the back, and slide it into a name tag holder with lanyard.

2.5 Tech

A typical full day camp will need the following tech supplies:

  • 30 chromebooks

  • 30 chargers

  • 30 mice

  • 8 power strips

  • 8 extention cords

2.6 Misc Supplies

Don’t forget to pack these too!

  • Market

  • ThoughtSTEM Dollars

  • Stickers (hearts and bonus)

  • Markers

  • 5 timers

The following items might also come in handy:

  • Extra tables

  • Combination lock for computer locker

  • Clipboards

  • Projector
