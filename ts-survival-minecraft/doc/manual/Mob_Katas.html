3.1 Mob Kata

read
Make a minecraft game with a basic-skinned hero and a mob.
code
#lang survival-minecraft
 
(minecraft-game 
    #:skin (custom-skin) 
    #:mob-list (list (custom-mob)))

3.2 Mob Kata

read
Make a minecraft game with a basic-skinned hero and 10 randomized mobs.
code
#lang survival-minecraft
 
(minecraft-game 
    #:skin (custom-skin) 
    #:mob-list (list (curry custom-mob 
    #:amount-in-world 10)))

3.3 Mob Kata

read
Make a minecraft game with a basic-skinned hero and 5 skeleton mobs with medium AI.
code
#lang survival-minecraft
 
(define (my-mob) (custom-mob 
    #:ai 'medium 
    #:sprite skeleton-sprite 
    #:amount-in-world 5))
 
(minecraft-game 
    #:skin (custom-skin) 
    #:mob-list (list (my-mob)))

3.4 Mob Kata

read
Make a minecraft game with a basic-skinned hero, 5 easy creeper mobs, and 5 medium skeleton mobs that only come out at night.
code
#lang survival-minecraft
 
(define (easy-mob) (custom-mob 
    #:ai 'easy 
    #:sprite creeper-sprite 
    #:amount-in-world 5))
 
(define (medium-mob) (custom-mob 
    #:ai 'medium 
    #:sprite skeleton-sprite 
    #:amount-in-world 5 
    #:night-only? #t))
 
(minecraft-game 
    #:skin (custom-skin) 
    #:mob-list (list (easy-mob) (medium-mob)))

3.5 Mob Kata

read
Make a minecraft game with a basic-skinned hero and 5 hard mobs that spit acid with 50 damage.
code
#lang survival-minecraft
 
(define (hard-mob) (custom-mob 
    #:ai 'hard 
    #:sprite skeleton-sprite 
    #:amount-in-world 5 
    #:weapon (acid-spitter 
    #:damage 50)))
 
(minecraft-game 
    #:skin (custom-skin) 
    #:mob-list (list (hard-mob)))
