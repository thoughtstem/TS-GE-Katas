9 Sky Katas

9.1 Sky Kata

read
Make a minecraft game with a basic-skinned hero and a slow changing sky.
code
#lang survival-minecraft
 
(minecraft-game
  #:skin (custom-skin)
  #:sky (custom-sky #:length-of-day 5000))

code

(minecraft-game
  #:skin (custom-skin)

9.4 Sky Kata

read
Make a minecraft game with a basic-skinned hero, a very short night-time, and a mob that only comes out at night.
code
#lang survival-minecraft
 
(minecraft-game
  #:skin (custom-skin)
  #:mob-list (list
   (custom-mob
    #:amount-in-world 20
    #:night-only? #t))
  #:sky (custom-sky
   #:length-of-day 2400
   #:start-of-daytime 200
   #:end-of-daytime 2200))
