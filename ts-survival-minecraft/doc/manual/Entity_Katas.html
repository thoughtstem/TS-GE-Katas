7.1 Entity Kata

read
Make a minecraft game with a basic-skinned hero and a basic entity.
code
#lang survival-minecraft
 
(minecraft-game
  #:skin (custom-skin)
  #:entity-list (list (custom-entity)))

7.2 Entity Kata

read
Make a minecraft game with a basic-skinned hero and a pig entity with a new name.
code
#lang survival-minecraft
 
(define (my-npc)
   (custom-entity
    #:sprite witch-sprite
    #:name "Witch"))
 
(minecraft-game
  #:skin (custom-skin)
  #:entity-list (list (my-npc)))

7.3 Entity Kata

read
Make a minecraft game with a basic-skinned hero and a pig entity with a custom name, tile location, and movement mode.
code
#lang survival-minecraft
 
(define (my-entity)
   (custom-entity
    #:sprite pig-sprite
    #:name "Sir Pigsnoot"
    #:tile 3
    #:mode 'follow))
 
(minecraft-game
  #:skin (custom-skin)
  #:entity-list (list (my-entity)))

7.4 Entity Kata

read
Make a minecraft game with a basic-skinned hero and an entity with custom dialog.
code
#lang survival-minecraft
 
(define (my-entity)
   (custom-entity
    #:dialog (list
     "Woah, who are you??"
     "Wait, I'm a chicken..."
     "I can't talk!")))
 
(minecraft-game
  #:skin (custom-skin)
  #:entity-list (list (my-entity)))

7.5 Entity Kata

read
Make a minecraft game with a basic-skinned hero and two entities with custom dialog, as well as two other customizations.
code
#lang survival-minecraft
 
(define (my-entity)
   (custom-entity
    #:name "Francis"
    #:tile 4
    #:dialog (list
     "Greetings!"
     "Gee, you look hungry."
     "I'd offer you some chicken feed..."
     "but I don't think you'd like it.")))
 
(define (another-entity)
   (custom-entity
    #:sprite pig-sprite
    #:name "Mr. Piggstockerton III"
    #:mode 'pace
    #:dialog (list
     "oink oink oink oink"
     "oink OINK OINK"
     "oooOIINK")))
 
(minecraft-game
  #:skin (custom-skin)
  #:entity-list (list (my-entity) (another-entity)))
