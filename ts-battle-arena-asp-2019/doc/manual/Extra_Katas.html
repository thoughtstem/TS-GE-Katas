6.1 Background Kata (difficulty = 1)

read
Make a game that has a custom background.
code
#lang battle-arena
 
(battle-arena-game #:bg (custom-bg))

6.2 Background Kata (difficulty = 1)

read
Make a game that uses a predefined image as the background.
code
#lang battle-arena
 
(define (my-bg) (custom-bg #:bg-img LAVA-BG))
 
(battle-arena-game #:bg (my-bg))

6.3 Background Kata (difficulty = 1)

read
Make a game that has a fully customized background.
code
#lang battle-arena
 
(define (my-bg)
   (custom-bg
    #:bg-img LAVA-BG
    #:rows 2
    #:columns 2
    #:start-tile 4))
 
(battle-arena-game #:bg (my-bg))

6.4 Rocket Tower Kata (difficulty = 1)

read
Make a game that has a custom weapon that deploys Rocket Tower.
code
#lang battle-arena
 
(battle-arena-game
  #:weapon-list (list
   (custom-weapon
    #:name "Rocket Tower"
    #:sprite (make-icon "RT")
    #:dart (rocket-tower-builder))))

6.5 Rocket Tower Kata (difficulty = 1)

read

code
#lang battle-arena

  #:weapon-list (list
   (custom-weapon
    #:name "Rocket Tower"
    #:sprite (make-icon "RT")

     #:speed 2
     #:fire-mode 'homing))))

6.11 Repeater Armor Kata (difficulty = 1)

read
Make a game that has working Repeater Armor.
code
#lang battle-arena
 
(battle-arena-game
  #:item-list (list
   (custom-armor
    #:name "Repeater Armor"
    #:sprite (make-icon "RA")
    #:protects-from "Repeater"
    #:change-damage (divide-by 2)
    #:rarity 'rare)))

6.12 Repeater Armor Kata (difficulty = 2)

read
Make a game that has 10 enemies with a repeater and armor for your avatar that will protect against that repeater!
code
#lang battle-arena
 
(battle-arena-game
  #:enemy-list (list
   (custom-enemy
    #:amount-in-world 10
    #:weapon (custom-weapon #:name "Repeater")))
  #:item-list (list
   (custom-armor
    #:name "Repeater Armor"
    #:sprite (make-icon "RA")
    #:protects-from "Repeater"
    #:change-damage (divide-by 2)
    #:rarity 'rare)))
