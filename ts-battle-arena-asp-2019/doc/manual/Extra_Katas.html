6.1 Background Kata (difficulty = 1)

read
Make a game that has a custom background.
code
#lang battle-arena
 
(battle-arena-game #:bg (custom-bg))

6.2 Background Kata (difficulty = 1)

Tip: See other bg-image options in the assets library.
read
Make a game that has a lava background.
code
#lang battle-arena
 
(battle-arena-game
  #:bg (custom-bg #:img LAVA-BG))

6.3 Background Kata (difficulty = 1)

Tip: Rows and columns default to 3x3.
read
Make a game that has a lava background and a 2 by 2 grid.
code
#lang battle-arena
 
(define (my-bg)
   (custom-bg #:img LAVA-BG #:rows 2 #:columns 2))
 
(battle-arena-game #:bg (my-bg))

6.4 Background Kata (difficulty = 1)

Tip: Start-tile can be between 0 (the default) and total-tiles minus 1. For example, a 2x2 grid has 4 tiles, so the last tile is tile 3.
read
Make a game that has a high definition lava background, 2 by 2 grid, and starts on tile 3.
code
#lang battle-arena
 
(define (my-bg)
   (custom-bg
    #:img LAVA-BG
    #:rows 2
    #:columns 2
    #:start-tile 3
    #:hd? #t))
 
(battle-arena-game #:bg (my-bg))

6.5 Level Design Kata (difficulty = 1)

Tip: This will fill the world with two types of trees.
read
Make a game with a forest background filled with trees.
code
#lang battle-arena
 
(battle-arena-game
  #:bg (custom-bg #:img FOREST-BG)
  #:enable-world-objects? #t)

6.6 Level Design Kata (difficulty = 1)

Tip: Setting hd? #t will use more detailed assets but it will take longer to load.
read
Make a game with a forest background filled with high definition trees.
code
#lang battle-arena
 
(battle-arena-game
  #:bg (custom-bg #:img FOREST-BG)
  #:other-entities (make-world-objects
   round-tree
   pine-tree
   #:hd? #t))

6.7 Level Design Kata (difficulty = 2)

Tip: With a 3x3 bg, the posn x value can be between 0 and 480 and the y between 0 and 360. Hue is any value between 0 and 360.
read
Make a game with 3 world objects with customized position, tile, size, and/or hue.
code
#lang battle-arena
 
(battle-arena-game
  #:other-entities (pine-tree (posn 100 200) #:tile 0)
  (wood-house (posn 100 200) #:tile 1 #:size 0.5)
  (brick-house
   (posn 100 200)
   #:tile 2
   #:hue (random 360)))

6.8 Level Design Kata (difficulty = 1)

Tip: make-world-objects works best with houses or trees and must only take two types. See the Assets Library for more options.
read
Make a game with a pink background filled with random color candy-cane-trees and snow-pine-trees.
code
#lang battle-arena
 
(battle-arena-game
  #:bg (custom-bg #:img PINK-BG)
  #:other-entities (make-world-objects
   candy-cane-tree
   snow-pine-tree
   #:hd? #t
   #:random-color? #t))

6.9 Repeater Armor Kata (difficulty = 1)

read
Make a game that has armor called Repeater Armor.
code
#lang battle-arena
 
(battle-arena-game
  #:item-list (list
   (custom-armor
    #:name "Repeater Armor"
    #:sprite (make-icon "RA"))))

6.10 Repeater Armor Kata (difficulty = 1)

read
Make a game that has working Repeater Armor.
code
#lang battle-arena
 
(battle-arena-game
  #:item-list (list
   (custom-armor
    #:name "Repeater Armor"
    #:sprite (make-icon "RA")
    #:protects-from "Repeater"
    #:change-damage (divide-by 2)
    #:rarity 'rare)))

6.11 Repeater Armor Kata (difficulty = 2)

read
Make a game that has 10 enemies with a repeater and armor for your avatar that will protect against that repeater!
code
#lang battle-arena
 
(battle-arena-game
  #:enemy-list (list
   (custom-enemy
    #:amount-in-world 10
    #:weapon (custom-weapon #:name "Repeater")))
  #:item-list (list
   (custom-armor
    #:name "Repeater Armor"
    #:sprite (make-icon "RA")
    #:protects-from "Repeater"
    #:change-damage (divide-by 2)
    #:rarity 'rare)))
