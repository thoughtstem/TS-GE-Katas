2.2 Avatar Kata (difficulty = 1)

read
Make a game that has a custom avatar with a sprite picked from the assets library.
code
#lang battle-arena
 
(battle-arena-game
  #:avatar (custom-avatar #:sprite pirateboy-sprite))

2.3 Avatar Kata (difficulty = 1)

Tip: The default speed is 10.
read
Make a game that has a custom avatar with a custom sprite and 20 speed.
code
#lang battle-arena
 
(define (my-avatar)
   (custom-avatar
    #:sprite pirategirl-sprite
    #:speed 20))
 
(battle-arena-game #:avatar (my-avatar))

2.4 Avatar Kata (difficulty = 1)

Tip: The default item-slots is 2.
read
Make a game that has a custom avatar with a custom sprite, 20 speed, and 5 item slots.
code
#lang battle-arena
 
(define (my-avatar)
   (custom-avatar
    #:sprite steampunkgirl-sprite
    #:speed 20
    #:item-slots 5))
 
(battle-arena-game #:avatar (my-avatar))
