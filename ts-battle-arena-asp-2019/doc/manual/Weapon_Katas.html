4 Weapon Katas

4.1 Sword Kata (difficulty = 1)

Tip: The default damage is 25
read
Make a game that has a sword that does 50 damage.
code
#lang battle-arena
 
(battle-arena-game
  #:weapon-list (list (sword #:name "Heavy Sword" #:damage 50)))

4.2 Spear Kata (difficulty = 2)

read
Make a game that has a ’blue ’legendary spear with 100 damage, 20 speed, and 10 range.
code
#lang battle-arena
 
(define (my-spear)
   (spear
    #:name "Spear"
    #:icon (make-icon "LS" 'blue)
    #:sprite (set-sprite-color 'blue spear-sprite)
    #:damage 100
    #:speed 20
    #:range 10
    #:rarity 'legendary))
 
(battle-arena-game
  #:weapon-list (list (my-spear)))

4.3 Fire Magic Kata (difficulty = 2)

Tip: You can also try ice-magic, sword-magic, ring-of-fire, ring-of-ice, or ring-of-blades.
read
Make a game that has fire magic with 20 damage.
code
#lang battle-arena
 
(define (my-weapon)
   (fire-magic
    #:name "Heavy Fire Magic"
    #:damage 20))
 
(battle-arena-game
  #:weapon-list (list (my-weapon)))

4.4 Fire Magic Kata (difficulty = 2)

Tip: If you use ring-of-fire, ring-of-ice, or ring-of-blades, you must change #:range to #:duration.
read
Make a game that has a ’legendary fire magic with a customized sprite, damage, speed, and range.
code
#lang battle-arena
 
(define (my-weapon)
   (fire-magic
    #:name "Fire Magic"
    #:icon (make-icon "FM" 'red)
    #:sprite (set-sprite-color 'green flame-sprite)
    #:damage 20
    #:speed 10
    #:range 50
    #:rarity 'legendary))
 
(battle-arena-game
  #:weapon-list (list (my-weapon)))

4.5 Single Shot Kata (difficulty = 1)

Tip: #:rapid-fire? defaults to #t. True (#t) means that you can hold the fire button to shoot.
read
Make a game with a weapon that fires only once each click.
code
#lang battle-arena
 
(battle-arena-game
  #:weapon-list (list
   (custom-weapon
    #:name "Single Shot"
    #:sprite (make-icon "SS")
    #:rapid-fire? #f)))

4.6 Spread Shot Kata (difficulty = 2)

Tip: Default damage, durability, and speed are 10, 10, and 10. A durabilty of 20 means each dart can hit 2 targets.
read
Make a game with a spread-shot weapon that fires customized darts that can go through one enemy and hit another.
code
#lang battle-arena
 
(define (my-dart)
   (custom-dart
    #:damage 20
    #:durability 20
    #:speed 15))
 
(battle-arena-game
  #:weapon-list (list
   (custom-weapon
    #:name "Spread Shot"
    #:sprite (make-icon "SPR")
    #:fire-mode 'spread
    #:dart (my-dart))))

4.7 Homing Repeater Kata (difficulty = 4)

Rarity Rules:

  • ’common = 5 total

  • ’uncommon = 4 total

  • ’rare = 3 total

  • ’epic = 2 total

  • ’legendary = 1 total

read
Make a game with a ’rare weapon that fires homing darts with customized sprite, damage, speed, and range.
code
#lang battle-arena
 
(define (my-dart)
   (custom-dart
    #:sprite (rectangle 10 2 'solid 'pink)
    #:damage 15
    #:speed 8
    #:range 40))
 
(define (homing-shot)
   (custom-weapon
    #:name "Homing Repeater"
    #:sprite (make-icon "HR")
    #:fire-mode 'homing
    #:dart (my-dart)
    #:rarity 'rare))
 
(battle-arena-game
  #:weapon-list (list (homing-shot)))
