10 Force Field Katas

10.1 Force Field Kata (force-field-1)

read
Make a game that has an item with a custom name that spawns a force field.
code
#lang battle-arena-fortnite
 
(fortnite-game
  #:item-list (list
   (custom-item
    #:name "Force Field"
    #:on-use (spawn (force-field)))))

10.2 Force Field Kata (force-field-2)

read
Make a game that has an item with a custom name and sprite, that spawns a force field with a custom duration.
code
#lang battle-arena-fortnite
 
(fortnite-game
  #:item-list (list
   (custom-item
    #:name "Force Field"
    #:sprite (make-icon "FF")
    #:on-use (spawn (force-field #:duration 1000)))))

10.3 Force Field Kata (force-field-3)

read
Make a game that has an item with a custom name and sprite, that spawns a force field with a custom duration, and that allows friendly darts to go through it.
code
#lang battle-arena-fortnite
 
(define (force-field-item)
   (custom-item
    #:name "Force Field"
    #:sprite (make-icon "FF")
    #:on-use (spawn
     (force-field
      #:allow-friendly-dart? #t
      #:duration 1000))))
 
(fortnite-game
  #:item-list (list (force-field-item)))
