10 Extra Katas

10.1 Air Starvation Rate Kata (3 minutes)

main This is an optinal Kata.

main The formula is (100 - x) = number of ticks to lose 1 health.

main If x = 99 the avatar will lose 1 health every tick. If x = 50 the avatar will lose 1 health every 50 ticks. If x = 0 the avatar will lose 1 health every 100 ticks. If x = -200 the avatar will lose 1 health every 300 ticks.

To earn this Kata students should be able to translate any sentence of this type:
Write a game with an avatar and a custom starvation rate.
To corresponding code of this type:
  #lang survival
  (survival-game
  #:avatar          (custom-avatar)
  #:starvation-rate 100)
