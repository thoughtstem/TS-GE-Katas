4 Avatar Katas

4.1 Air Avatar Kata (1 minutes)

main core values

main lively discussion

main This is a good kata to learn as a team or in small groups.

To earn this Kata students should be able to recite the amount of points they get from having a custom Avatar in their game.
Avatar points:
* Having an avatar - 1 point.
* Any avatar within 5 minutes (sprint bonus) - 5 points.
* Avatar animation frames (max 4 frames) - 4 points.

4.2 Bronze Avatar Kata (5 minutes)

To earn this Kata students should be able to translate any sentence of this type:
Write the shortest program that creates a game with an avatar.
To corresponding code of this type:
  #lang survival
  (survival-game
   #:avatar (custom-avatar))
   

4.3 Silver Avatar Kata (5 minutes)

To earn this Kata students should be able to translate any sentence of this type:
Make a game that has a different colored shape as an avatar.
To corresponding code of this type:
  #lang survival
  (define (my-avatar)
   (custom-avatar #:sprite (star 30 'solid 'yellow)))
   
  (survival-game
   #:avatar (my-avatar))
   

4.4 Gold Avatar Kata (5 minutes)

main Not including drawing time, but do include exporting time.

main Replace bat-sprite with your actual image.

To earn this Kata students should be able to translate any sentence of this type:
Make a game that has your drawing as the avatar.
To corresponding code of this type:
  #lang survival
  (define (my-avatar)
   (custom-avatar #:sprite bat-sprite))
   
  (survival-game
   #:avatar (my-avatar))
