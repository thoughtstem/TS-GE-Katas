6 Food Katas

6.1 Air Food Kata (1 minutes)

Review/Introduce: core values

Extra Dollars for: lively discussion

Tip: This is a good kata to learn as a team or in small groups.

To earn this Kata students should be able to recite the amount of points they get from having a custom Food in their game.
Food points:
* Each food item (max 10) - 1 point.
* Any food item within 15 minutes (sprint bonus) - 5 points.
* For each food item matching the theme (max 10) - 1 point.

6.2 Bronze Food Kata (5 minutes)

Tip: Students should be encouraged to add other game attributes; e.g. coins.

To earn this Kata students should be able to translate any sentence of this type:
Write the shortest program that creates a game with an avatar and a food item.
To corresponding code of this type:
  #lang survival
  (survival-game
  #:avatar     (custom-avatar)
  #:food-list  (list (custom-food #:amount-in-world 10)))
   

6.3 Silver Food Kata (5 minutes)

To earn this Kata students should be able to translate any sentence of this type:
Make a game that has an avatar and a food that heals by 20 points.
To corresponding code of this type:
  #lang survival
  (define (my-food)
   (custom-food #:amount-in-world 2
                #:heals-by        20))
   
  (survival-game
  #:avatar     (custom-avatar)
  #:food-list  (list (my-food)))
   

6.4 Gold Food Kata (5 minutes)

Tip: Students should run the function (show-food-sprites) on the interactions window to see a list of coin sprites.

Tip: The order of the keywords does not matter.

To earn this Kata students should be able to translate any sentence of this type:
Make a game that has an avatar and a food item with a custom sprite, name, amount in world, and healing power.
To corresponding code of this type:
  #lang survival
  (define (my-food)
   (custom-food #:sprite          apples-sprite
                #:name            "Apples"
                #:amount-in-world 2
                #:heals-by        20))
   
  (survival-game
  #:avatar     (custom-avatar)
  #:food-list  (list (my-food)))
   

6.5 Platinum Food Kata (10 minutes)

Tip: Any mix of keywords here is okay, as long as they all have the required sprite and name, and at least one of them does not respawn.

Tip: There is no number of "required" katas by day.

To earn this Kata students should be able to translate any sentence of this type:
Make a game that has an avatar and more than one food item with a custom sprite, name, amount in world, healing power, and respawn value.
To corresponding code of this type:
  #lang survival
  (define (my-food)
   (custom-food #:sprite          apples-sprite
                #:name            "Apple"
                #:amount-in-world 2
                #:heals-by        20))
   
  (define (special-food)
   (custom-food #:sprite          cherry-sprite
                #:name            "Cherry"
                #:amount-in-world 1
                #:heals-by        50
                #:respawn?        #f))
   
  (survival-game
  #:avatar     (custom-avatar)
  #:food-list  (list (my-food)
                     (special-food)))
