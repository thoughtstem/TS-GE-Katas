3 Tower Katas

3.1 Bronze Spear Tower Kata

When given this English phrase: Make a game that has a custom weapon that deploys a spear tower.
You can write this battle-arena code:
#lang battle-arena
(battle-arena-game
#:weapon-list (list (custom-weapon
                     #:name "Spear Tower"
                     #:sprite (make-icon "ST")
                     #:dart (spear-tower-builder))))
 

3.2 Silver Spear Tower Kata
