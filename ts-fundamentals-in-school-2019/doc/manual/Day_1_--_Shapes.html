1.1 Core Values Kata

hear
What are our core values?
say
It’s not about my code; it’s about what I can code!
It’s not about me; it’s about my team!

1.2 Hello World Kata

read
Make a circle that is red and size 40.
code
#lang racket
 
(require 2htdp/image)
 
(circle 40 'solid 'red)

1.3 Another Shape Kata

read
Make a star that is gold and size 60 pixels.
code
#lang racket
 
(require 2htdp/image)
 
(star 60 'solid 'gold)

1.4 Favorite Color Kata

read
Make a triangle of your favorite color and size 100.
code
#lang racket
 
(require 2htdp/image)
 
(triangle 100 'solid 'skyblue)

1.5 Not Solid? Kata

read
Write five copies of the line below, filling in the blank with a word that DOESN’T work! (circle 50 ’_______ ’red)
code
#lang racket
 
(require 2htdp/image)
 
(circle 30 'blue 'red)
(circle 30 'dotted 'red)
(circle 30 'line 'red)

1.6 Outline Shape Kata

Make a triangle of size 150 with a purple outline.
code
#lang racket
 
(require 2htdp/image)
 
(triangle 150 'outline 'purple)
