6.1 Crafter Kata

read
Make a game with a Pokemon and a basic crafter.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:crafter-list (list (custom-crafter)))

6.2 Crafter Kata

Tip: By adding a recipe, the crafter now requires an item in the avatar’s backpack to build something.

read
Make a game with a Pokemon, carrots, and a crafter that makes carrot stew from carrots.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:food-list (list (carrot #:amount-in-world 10))
  #:crafter-list (list
   (custom-crafter
    #:recipe-list (list carrot-stew-recipe))))

6.3 Crafter Kata

read
Make a game with a Pokemon, fish, and a crafter that makes fish stew that heals you by 50.
code
#lang survival-pokemon
 
(define (fish-stew)
   (custom-food
    #:name "Fish Stew"
    #:sprite fish-stew-sprite
    #:respawn? #f
    #:heals-by 50))
 
(define fish-stew-recipe
   (recipe
    #:product (fish-stew)
    #:build-time 40
    #:ingredients (list "Fish")))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:food-list (list (fish #:amount-in-world 10))
  #:crafter-list (list
   (custom-crafter
    #:sprite cauldron-sprite
    #:recipe-list (list fish-stew-recipe))))

6.4 Crafter Kata

read
Make a game with a Pokemon, carrots, fish, and a crafter on tile 2 (posn 200 200) that makes both carrot-stew and fish-stew.
code
#lang survival-pokemon
 
(define (fish-stew)
   (custom-food
    #:name "Fish Stew"
    #:sprite fish-stew-sprite
    #:respawn? #f
    #:heals-by 50))
 
(define fish-stew-recipe
   (recipe
    #:product (fish-stew)
    #:build-time 40
    #:ingredients (list "Fish")))
 
(define (my-cauldron)
   (custom-crafter
    #:sprite cauldron-sprite
    #:position (posn 200 200)
    #:tile 2
    #:recipe-list (list carrot-stew-recipe fish-stew-recipe)))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:food-list (list
   (carrot #:amount-in-world 10)
   (fish #:amount-in-world 10))
  #:crafter-list (list (my-cauldron)))
