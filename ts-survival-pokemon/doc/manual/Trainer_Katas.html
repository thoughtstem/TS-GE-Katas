5.1 Trainer Kata

read
Make a game with a Pokemon and a trainer.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:trainer-list (list (custom-trainer)))

5.2 Trainer Kata

Tip: Without the curry, the game will have 10 of the same enemy. With curry, the game will have 10 different enemies.

read
Make a game with a Pokemon and 10 random trainers.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:trainer-list (list
   (curry custom-trainer #:amount-in-world 10)))

5.3 Trainer Kata

read
Make a game with a Pokemon and 5 hard Red Girl trainers that throw pokeballs with 50 damage.
code
#lang survival-pokemon
 
(define (hard-trainer)
   (custom-trainer
    #:ai 'hard
    #:sprite redgirl-sprite
    #:amount-in-world 5
    #:weapon (pokeball #:damage 50)))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:trainer-list (list (hard-trainer)))

5.4 Trainer Kata

Tip: The options for #:ai are ’easy, ’medium and ’hard.

read
Make a game with a Pokemon and 5 James with medium AI.
code
#lang survival-pokemon
 
(define (my-trainer)
   (custom-trainer
    #:ai 'medium
    #:sprite james-sprite
    #:amount-in-world 5))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:trainer-list (list (my-trainer)))

5.5 Trainer Kata

read
Make a game with a Pokemon, 5 easy Green Boy trainers, and 5 medium Green Girl trainers that only come out at night.
code
#lang survival-pokemon
 
(define (easy-trainer)
   (custom-trainer
    #:ai 'easy
    #:sprite greenboy-sprite
    #:amount-in-world 5))
 
(define (medium-trainer)
   (custom-trainer
    #:ai 'medium
    #:sprite greengirl-sprite
    #:amount-in-world 5
    #:night-only? #t))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:trainer-list (list (easy-trainer) (medium-trainer)))
