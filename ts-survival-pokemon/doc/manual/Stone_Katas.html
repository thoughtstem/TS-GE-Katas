4.1 Stone Kata

read
Make a game with a Pokemon and a basic collectible stone.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:stone-list (list (custom-stone)))

4.2 Stone Kata

Tip: The default value of custom-coin is 10.

read
Make a game with a Pokemon and a collectible stone worth 500.
code
#lang survival-pokemon
 
(define (my-stone) (custom-stone #:value 500))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:stone-list (list (my-stone)))

4.3 Stone Kata

read
Make a game with a Pokemon and 20 moon stones worth 500.
code
#lang survival-pokemon
 
(define (my-stone)
   (custom-stone
    #:sprite moonstone-sprite
    #:name "Moon Stone"
    #:value 500
    #:amount-in-world 20))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:stone-list (list (my-stone)))

4.4 Stone Kata

read
Make a game with a Pokemon, a moon stone, and a single shiny stone that never respawns and is worth 1000.
code
#lang survival-pokemon
 
(define (my-stone)
   (custom-stone
    #:sprite moonstone-sprite
    #:name "Moon Stone"))
 
(define (my-special-stone)
   (custom-stone
    #:sprite shinystone-sprite
    #:name "Shiny Stone"
    #:value 1000
    #:amount-in-world 1
    #:respawn? #f))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:stone-list (list (my-stone) (my-special-stone)))
