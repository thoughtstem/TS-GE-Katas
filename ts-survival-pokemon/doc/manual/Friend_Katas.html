7 Friend Katas

7.1 Friend Kata

read
Make a game with a Pokemon and another Pokemon friend.
code
#lang survival-pokemon
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:friend-list (list (custom-friend)))

7.2 Friend Kata

read
Make a game with a Pokemon and Squirtle with a custom name.
code
#lang survival-pokemon
 
(define (my-friend)
   (custom-friend
    #:sprite squirtle-sprite
    #:name "Torty"))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:friend-list (list (my-friend)))

7.3 Friend Kata

read
Make a game with a Pokemon and another Pokemon with a name, starting tile, and follow mode.
code
#lang survival-pokemon
 
(define (my-friend)
   (custom-friend
    #:sprite wartortle-sprite
    #:name "Torts"
    #:tile 3
    #:mode 'follow))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:friend-list (list (my-friend)))

7.4 Friend Kata

Tip: All options for #:mode are: ’follow, ’wander, ’pace and ’still.

read
Make a game with a Pokemon and a friend with custom dialog.
code
#lang survival-pokemon
 
(define (my-friend)
   (custom-friend
    #:dialog (list
     "Woah, who are you??"
     "Nevermind -- I'm too busy."
     "Move along, now!")))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:friend-list (list (my-friend)))

7.5 Friend Kata

read
Make a game with a Pokemon and two friends with custom dialog and two other customizations.
code
#lang survival-pokemon
 
(define (my-friend)
   (custom-friend
    #:name "Francis"
    #:tile 4
    #:dialog (list
     "Greetings!"
     "You better find some food soon...")))
 
(define (another-friend)
   (custom-friend
    #:sprite ivysaur-sprite
    #:mode 'pace
    #:dialog (list
     "Now where did I put it..."
     "Have you seen a leaf stone?"
     "Oh, I think I see it!")))
 
(pokemon-game
  #:pokemon (custom-pokemon)
  #:friend-list (list (my-friend) (another-friend)))
