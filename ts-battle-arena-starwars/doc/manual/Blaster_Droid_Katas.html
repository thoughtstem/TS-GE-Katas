13 Blaster Droid Katas

13.1 Blaster Droid Kata (blaster-droid-1)

read
Make a game that has a weapon that spawns a droid with a blaster.
code
#lang battle-arena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Blaster Droid"
    #:sprite (make-icon "BD")
    #:dart (blaster-droid))))

13.2 Blaster Droid Kata (blaster-droid-2)

read
Make a game that has a defined weapon that spawns a droid with a customized blaster.
code
#lang battle-arena-starwars
 
(define (b-droid)
   (custom-weapon
    #:name "Blaster Droid"
    #:sprite (make-icon "BD")
    #:dart (blaster-droid
     #:color "yellow"
     #:damage 25
     #:fire-mode 'spread)))
 
(starwars-game #:weapon-list (list (b-droid)))

13.3 Blaster Droid Kata (blaster-droid-3)

read
Make a game that has a defined weapon that spawns a droid with a fully customized blaster.
code
#lang battle-arena-starwars
 
(define (b-droid)
   (custom-weapon
    #:name "Blaster Droid"
    #:sprite (make-icon "BD")
    #:dart (blaster-droid
     #:color "yellow"
     #:damage 15
     #:durability 25
     #:speed 10
     #:range 75
     #:fire-rate 5
     #:fire-mode 'homing)))
 
(starwars-game #:weapon-list (list (b-droid)))
