10 Shield Katas

10.1 Shield Kata (shield-1)

read
Make a game that has an powerup that increases your shield by 50.
code
#lang battle-arena-starwars
 
(starwars-game
  #:item-list (list
   (custom-item
    #:name "Shield Powerup"
    #:sprite (make-icon "SP" 'blue 'white)
    #:on-use (change-shield-by 50))))

10.2 Shield Kata (shield-2)

read
Make a game that has an powerup that recovers your shield to 100.
code
#lang battle-arena-starwars
 
(starwars-game
  #:item-list (list
   (custom-item
    #:name "Max Shield Powerup"
    #:sprite (make-icon "MSP" 'blue 'white)
    #:on-use (set-shield-to 100))))

10.3 Shield Kata (shield-3)

read
Make a game that has both a Shield and Max Shield defined powerup.
code
#lang battle-arena-starwars
 
(define (shield-powerup)
   (custom-item
    #:name "Shield Powerup"
    #:sprite (make-icon "SP" 'blue 'white)
    #:on-use (change-shield-by 50)
    #:rarity 'uncommon
    #:respawn? #t))
 
(define (max-shield)
   (custom-item
    #:name "Max Shield Powerup"
    #:sprite (make-icon "MSP" 'blue 'white)
    #:on-use (set-shield-to 100)
    #:rarity 'epic))
 
(starwars-game
  #:item-list (list (shield-powerup) (max-shield)))
