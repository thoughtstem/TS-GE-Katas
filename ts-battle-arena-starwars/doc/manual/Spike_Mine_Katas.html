17 Spike Mine Katas

17.1 Spike Mine Kata (spike-mine-1)

read
Make a game that has a weapon that deploys a default spike mine
code
#lang battle-arena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Spike Mine"
    #:sprite (make-icon "SM")
    #:dart (spike-mine-builder))))

17.2 Spike Mine Kata (spike-mine-2)

read
Make a game that has a weapon that deploys a spike mine with a large range and speed
code
#lang battle-arena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Spike Mine"
    #:sprite (make-icon "SM" 'gray)
    #:dart (spike-mine-builder #:speed 10 #:range 50))))

17.3 Spike Mine Kata (spike-mine-3)

read
Make a game that has a weapon that deploys a spike mine with a large range and speed, high damage, and non-default sprite
code
#lang battle-arena-starwars
 
(starwars-game
  #:weapon-list (list
   (custom-weapon
    #:name "Spike Mine"
    #:sprite (make-icon "SM" 'gray 'white)
    #:dart (spike-mine-builder
     #:sprite (set-sprite-color 'red spike-mine-sprite)
     #:damage 100
     #:speed 10
     #:range 50))))
