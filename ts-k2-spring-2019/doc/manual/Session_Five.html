5.1 Warm Ups Asl Kata

read
#lang k2/lang/classroom/basic
 
(whenever (hear 'A) (sign 'A))
 
(whenever (hear 'B) (sign 'B))
 
(whenever (hear 'C) (sign 'C))
 
(whenever (hear 'D) (sign 'D))
In Ratchet:
do
The social game above as a class until everyone can play accurately and efficiently. As with all games, the teacher may adjust the rules at any time to customize the game for their class.

5.2 Bots Kata

read
#lang k2/lang/classroom/basic
 
(whenever (see 'A) (turn 2))
 
(whenever (see 'B) (turn 3))
 
(whenever (see 'C) (step 1))
 
(whenever (see 'D) (step 2))
In Ratchet:
do
The social game above as a class until everyone can play accurately and efficiently. As with all games, the teacher may adjust the rules at any time to customize the game for their class.

5.3 Hero Kata

read
Write code to make a a game where Ironman is the hero, and there are no enemies. Give the hero a red Energy Blast power.
code
#lang k2/lang/hero/powers
 
(start (ironman (energy-blast red)))

5.4 Hero Kata

read
Write code to make a game where Ironman is the hero, and three Gamoras are your enemies. Give ONE Gamora a Hammer power.
code
#lang k2/lang/hero/powers
 
(start ironman (gamora hammer) gamora gamora)
