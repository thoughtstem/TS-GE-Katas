7.1 Bots Kata

read
#lang k2/lang/classroom/basic
 
(assign
  'Bots
  (whenever (hear 'A) (step 1))
  (whenever (hear 'B) (turn 2)))
 
(assign
  'Control
  (whenever (see 'C) (say 'A))
  (whenever (see 'D) (say 'B)))
In Ratchet:
do
The social game above as a class until everyone can play accurately and efficiently. As with all games, the teacher may adjust the rules at any time to customize the game for their class.

7.4 Hero Kata

read
Write code to make a a game with three levels. In the first level, the enemy is Gamora, in the second level the enemy is Drax, in the third level the enemy is Hawkeye. In all levels, the hero is Ironman.
code
#lang k2/lang/hero/powers
 
(start ironman gamora)
 
(start ironman drax)
 
(start ironman hawkeye)
