4 Picture Project Katas

4.1 Bronze Piture Project Kata (5 minutes)

A.K.A. House Kata

To earn this Kata student should build a simple house picture.

For example:

#lang racket
(require 2htdp/image)
 
(above
 (triangle 100 'solid 'aqua)
 (square 100 'solid 'blue))

4.2 Silver Picture Project Kata (5 minutes)

A.K.A. Tree Kata

To earn this Kata student should build a simple tree picture.

Example complete code:

#lang racket
(require 2htdp/image)
 
(above
 (circle 30 'solid 'green)
 (rectangle 20 80 'solid 'brown))

4.3 Gold Picture Project Kata (10 minutes)

A.K.A. House Plus Kata

To earn this Kata student should build a simple house picture, adding a door.

Teaching Tip: Spend some time messing around with overlay/align. Options for the first parameter, "middle", also include "right" and "left." "bottom" can also be "middle" or "top"

Example complete code:

#lang racket
(require 2htdp/image)
 
(above
 (triangle 100 'solid 'aqua)
 
 (overlay/align "middle" "bottom"
                (rectangle 30 50 'solid 'brown)
                (square 100 'solid 'blue)))

4.4 Platinum Picture Project Kata (10 minutes)

A.K.A. House Set Kata

To earn this Kata student should build a house with a door beside

Teaching Tip: You may want to start with the complete code from the last kata, then add the new code (focusing on relationships and parentheses), before you start building it all from scratch.

Example complete code:

#lang racket
(require 2htdp/image)
 
(beside/align "bottom"
 (above
  (triangle 100 'solid 'aqua)
 
  (overlay/align "middle" "bottom"
                 (rectangle 30 50 'solid 'brown)
                 (square 100 'solid 'blue)))
 
 (above
  (circle 30 'solid 'green)
  (rectangle 20 80 'solid 'brown)))
