5.2 Spear Kata (spear-2)

Tip: The default damage is 25
read
Make a game that has a spear with 50 damage.
code
#lang battle-arena
 
(battle-arena-game
  #:weapon-list (list (spear #:name "Heavy Spear" #:damage 50)))

5.3 Spear Kata (spear-4)

read
Make a game that has a ’blue ’legendary spear with 100 damage, 20 speed, and 10 range.
code
#lang battle-arena
 
(define (my-spear)
   (spear
    #:name "Spear"
    #:icon (make-icon "LS" 'blue)
    #:sprite (set-sprite-color 'blue spear-sprite)
    #:damage 100
    #:speed 20
    #:range 10
    #:rarity 'legendary))
 
(battle-arena-game
  #:weapon-list (list (my-spear)))

5.4 Spear Kata (spear-3)

Rarity Rules:

  • ’common = 5 total

  • ’uncommon = 4 total

  • ’rare = 3 total

  • ’epic = 2 total

  • ’legendary = 1 total

read
Make a game that has an ’epic rarity spear with 50 damage.
code
#lang battle-arena
 
(define (my-spear)
   (spear
    #:name "Heavy Spear"
    #:damage 50
    #:rarity 'epic))
 
(battle-arena-game
  #:weapon-list (list (my-spear)))
