20 Boost Katas

20.1 Boost Kata (boost-1)

read
Make a game that has a powerup that temporarily boosts your health.
code
#lang battle-arena
 
(battle-arena-game
  #:item-list (list
   (custom-item
    #:name "Damage Boost"
    #:sprite (make-icon "DB" 'orangered)
    #:on-use (change-damage-by 1000 #:for 200))))

20.2 Boost Kata (boost-2)

read

#lang battle-arena
 
(battle-arena-game

20.3 Boost Kata (boost-3)

read
Make a game that has both a Damage and a Speed Boost powerup, defined outside battle-arena-game.
code
#lang battle-arena
 
(define (damage-boost)
   (custom-item
    #:name "Damage Boost"
    #:sprite (make-icon "DB" 'orangered)
    #:on-use (change-damage-by 1000 #:for 200)
    #:rarity 'epic))
 
(define (speed-boost)
   (custom-item
    #:name "Speed Boost"
    #:sprite (make-icon "SB" 'yellow)
    #:on-use (multiply-speed-by 2 #:for 200)
    #:rarity 'uncommon))
 
(battle-arena-game
  #:item-list (list (damage-boost) (speed-boost)))
