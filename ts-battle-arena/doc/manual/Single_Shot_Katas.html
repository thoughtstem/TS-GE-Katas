18 Single Shot Katas

18.1 spear-tower-1

read
Make a game that has a custom weapon that deploys a spear tower.
code
#lang battle-arena
(battle-arena-game
#:weapon-list (list (custom-weapon
                     #:name "Spear Tower"
                     #:sprite (make-icon "ST")
                     #:dart (spear-tower-builder))))
 

18.2 spear-tower-2

read
Make a game that has a custom weapon that deploys a spear tower with a speed of 10 and a range of 20.
code
#lang battle-arena
(battle-arena-game
#:weapon-list (list (custom-weapon
                     #:name "ST"
                     #:sprite (make-icon "ST" 'tan)
                     #:dart (spear-tower-builder
                             #:speed 10
                             #:range 50))))
 

18.3 spear-tower-3

read
Make a game that has a custom weapon that deploys a spear tower with a custom spear image, damage level, range and speed
code
#lang battle-arena
(battle-arena-game
#:weapon-list (list (custom-weapon
                     #:name "Spear Tower"
                     #:sprite (make-icon "ST" 'tan 'white)
                     #:dart (spear-tower-builder
                             #:sprite STUDENT-IMAGE-HERE
                             #:damage 100
                             #:speed 10
                             #:range 50))))
