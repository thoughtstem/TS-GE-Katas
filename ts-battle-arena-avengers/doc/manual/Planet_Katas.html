6.3 Planet Kata (planet-3)

read
Make a game that has a lava background, 2 by 2 grid, and starts on tile 4.
code
#lang battle-arena-avengers
 
(define (my-bg)
   (custom-bg
    #:img LAVA-BG
    #:rows 2
    #:columns 2
    #:start-tile 4))
 
(avengers-game #:planet (my-bg))
