code

(avengers-game

2.3 Hero Kata (hero-3)

Tip: The default key-mode is ’wasd.
read
Make a game with a fully customized hero.
code
#lang battle-arena-avengers
 
(define (my-hero)
   (custom-hero
    #:sprite thor-sprite
    #:speed 15
    #:key-mode 'arrow-keys
    #:item-slots 5))
 
(avengers-game #:hero (my-hero))
