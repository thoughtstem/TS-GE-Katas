13 Lava Pit Katas

13.1 Lava Pit Kata (lava-pit-1)

read
Make a game that has a custom weapon that deploys a lava pit.
code
#lang battle-arena-avengers
 
(avengers-game
  #:power-list (list
   (custom-power
    #:name "Lava Pit"
    #:sprite (make-icon "LP")
    #:dart (lava-builder))))

13.2 Lava Pit Kata (lava-pit-2)

read
Make a game that has a custom weapon that deploys a lava pit with custom damage and size.
code
#lang battle-arena-avengers
 
(avengers-game
  #:power-list (list
   (custom-power
    #:name "Lava Pit"
    #:sprite (make-icon "LP" 'red)
    #:dart (lava-builder #:damage 25 #:size 2))))

13.3 Lava Pit Kata (lava-pit-3)

read
Make a game that has a custom weapon that deploys a lava pit with custom damage, size, sprite, and range.
code
#lang battle-arena-avengers
 
(define (my-lava-pit)
   (custom-power
    #:name "Lava Pit"
    #:sprite (make-icon "LP" 'red 'white)
    #:dart (lava-builder
     #:damage 25
     #:size 1
     #:sprite (square 10 'solid 'black)
     #:range 10)))
 
(avengers-game #:power-list (list (my-lava-pit)))
