3.2 Villain Kata (villain-2)

read
Make a game with 10 random enemies
code
#lang battle-arena-avengers
 
(avengers-game
  #:villain-list (list
   (curry custom-villain #:amount-in-world 10)))

3.3 Villain Kata (villain-3)

read
Make a game that has 5 enemies with a darkknight sprite and ’medium ai.
code
#lang battle-arena-avengers
 
(define (my-enemy)
   (custom-villain
    #:sprite loki-sprite
    #:ai 'medium
    #:health 200
    #:shield 100
    #:amount-in-world 5))
 
(avengers-game #:villain-list (list (my-enemy)))

3.4 Villain Kata (villain-4)

read
Make a game that has 5 ’easy wizard enemies with 50 health and 5 ’medium darkknight enemies with 200 health.
code
#lang battle-arena-avengers
 
(define (easy-enemy)
   (custom-villain
    #:ai 'easy
    #:sprite wizard-sprite
    #:health 50
    #:amount-in-world 5))
 
(define (medium-enemy)
   (custom-villain
    #:ai 'medium
    #:sprite darkknight-sprite
    #:health 200
    #:amount-in-world 5))
 
(avengers-game
  #:villain-list (list (easy-enemy) (medium-enemy)))

3.5 Villain Kata (villain-5)

read
Make a game that has 5 ’hard pirategirl enemies using a custom-weapon with 50 damage.
code
#lang battle-arena-avengers
 
(define (hard-enemy)
   (custom-villain
    #:ai 'hard
    #:sprite pirategirl-sprite
    #:amount-in-world 5
    #:power (custom-power #:damage 50)))
 
(avengers-game
  #:hero (custom-hero)
  #:villain-list (list (hard-enemy)))
